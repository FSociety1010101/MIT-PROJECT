# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please report it by opening an issue in this repository with the label `security`.
Do not include proof-of-concept exploit code in a public issue.

## Supported Versions

- `main` branch: active development and security fixes.

## Response Policy

- We acknowledge reports within 48 hours.
- Critical issues will be addressed with a fix or mitigation as quickly as possible.

## External Dependencies

This project uses:
- Python 3.11+
- FastAPI
- tree-sitter Solidity parser
- React + Vite

Ensure dependencies are kept updated and pinned by reviewing `backend/requirements.txt` and `frontend/package.json`.
