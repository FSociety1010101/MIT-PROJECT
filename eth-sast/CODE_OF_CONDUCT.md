# Code of Conduct

## Our Pledge

In the interest of fostering an open and welcoming project, we pledge to respect all people who contribute to the project.

## Expected Behavior

- Be considerate and collaborative.
- Use welcoming language.
- Accept constructive feedback gracefully.
- Focus on what is best for the community.

## Unacceptable Behavior

- Harassment, discrimination, or offensive language.
- Personal attacks or demeaning comments.
- Disrespect toward others based on identity or background.

## Reporting

If you experience or witness unacceptable behavior, please open an issue or contact the repository maintainers.

## License

This Code of Conduct is adopted from the Contributor Covenant. By participating, you agree to abide by this Code of Conduct.
